# 職務経歴書 - 河田智治

2023/03/08　現在

## 職務要約
- テクニカル コンサルタントとして Microsoft のパートナー企業向けにコンサルテーションを提供。クラウド サービスである Microsoft Azure の Platform as a Service (PaaS) や、Microsoft Teams のボットなどを用いた クラウド ネイティブな Web アプリケーション開発関連の技術情報を案内。
- テクニカル コンサルタントとして Microsoft のパートナー企業向けにコンサルテーションを提供。Microsoft Power Platform のうち Power Apps、Power Automate、Power Virtual Agents、Power Pages に関する開発関連の技術情報を案内。
- iOS デベロッパーとして iOS や macOS ビルトインの音声認識技術の自動テストを行うアプリケーション開発プロジェクトに参加。iOS や macOS のアプリケーションに対する新規機能追加やバグ修正などを担当。
- iOS デベロッパーとして IoT 機器と iOS アプリケーションを用いた子供を見守るサービスのアプリケーション開発プロジェクトに参加。IoT 機器から送られる位置情報や接続状況などのアプリケーション上への表示や、新規機能の追加を担当。
- Android デベロッパーとして Android 端末をテスト用の API サーバーとして動作させるプロジェクトへ参加。小規模のプロジェクトであったため全肯定を担当。
- Android デベロッパーとしてお客様がカスタマイズして利用している Android に対し、脆弱性のあるコードを修正するプロジェクトに参加。Android のライブラリや Linux Kernel などの脆弱性に対するセキュリティ パッチの作成を担当。

#### パートナー テクニカル コンサルタント (2021 年 7 月 - 現在)

- ロール概要

    - パートナー企業のビジネス フェーズや課題内容をヒアリング後、状況に応じた技術ドキュメントをベースとした技術情報の提供。
    - Microsoft Azure の PaaS サービスを主に担当し、Web アプリケーションのサーバーレス アーキテクチャなどの情報を提供。

- 業務スキル
    - パートナー企業への電話/Teams 会議を用いたビジネス フェーズの課題内容のヒアリング
    - パートナー企業への電話/メール/Teams 会議による技術ドキュメントをベースとした情報提供。Teams 会議の場合は 30 - 90 分のセッションにて Azure へアプリケーションのデプロイするなどのデモに加え、QA 対応なども提供。
    - 言語
        - 日本語: 電話/メール/Teams 会議によるパートナー企業の担当者へのヒアリングと情報提供、デモなどを行えるレベル。
        - 英語: 非日本語ネイティブ、非英語ネイティブである上司やチーム メンバーなど社内のメンバーとのコミュニケーションや技術的ディスカッション、協業を行えるレベル。

- 技術スキル
    - クラウド サービス 
        - Microsoft Azure: Azure App Service, Azure Functions, Azure Container Instances, Azure Kubernetes Service, Azure DevOps, etc...
        - Microsoft Power Platform: Power Apps, Power Automate, Power BI, Power Pages, Power Virtual Agents
        - GitHub
    - 開発スキル (技術ドキュメントにあるサンプル コードの内容を理解し、Azure へデプロイし動作させられるレベル)
        - 開発言語: C#, PHP, Node.js, etc 
        - コンテナー: Docker コンテナー (Linux、Windows), Kubernetes
    - プラットフォーム、ツール
        - エディタ/統合開発環境: Visual Studio Code, Visual Studio
        - shell: Zsh, Bash, PowerShell
        - ソースコード管理: git
        - コンテナー: Docker, Minikube (Kubernetes)

#### iOS デベロッパー - 音声認識技術テスト オートメーション プロジェクト (2020 年 12 月 - 2021 年 6 月)
- ロール概要
    - 顧客社内ツールである iOS や macOS に含まれる音声認識技術のテスト自動化アプリケーションのメンテナンス業務。テスターによるチケット ベースのバグ修正リクエストまたは新規機能追加リクエストが発行され、8 人のチームでチケットを分担し開発を行う。
    - リクエスト内容を把握し、チーム内でどのように修正を行うかをディスカッション後、修正を行う。新規機能の場合はテスターにどのような機能や動作が期待されているかを詳細にヒアリングし、実装を行う。
    - 顧客社内の関連チームとの協業によるテスト デバイス ラボの設営

- 業務のスキル
    - テスターがリクエストしたチケットベースのリクエストに対し、どのような修正または機能を追加するかをチームメンバー内でディスカッションし、理解。
    - 既存コード上に実装してあるアーキテクチャを理解し、既存関数の修正や新規関数の作成などを要件に合わせて実装。
    - テスターとコミュニケーションによる期待される動作のヒアリング。
    - 言語
        - 英語: 英語ネイティブである上司やチーム メンバーをはじめ、様々なチームに所属する社内のメンバーとのコミュニケーションや技術的ディスカッション、協業を行えるレベル。

- 技術スキル
    - OS: iOS (12, 13, 14), macOS (10.15, 11.1)

- 開発スキル
    - 開発言語: Objective-C
    - 開発モデル: アジャイル

- プラットフォーム、ツール
    - エディタ/統合開発環境: Xcode, Vim
    - shell: Zsh
    - ソースコード管理: git

- ロール概要
    - 受託開発による顧客サービス (Bluetooth, eSIM, GPS などのモジュールを持つ iOT 機器と iOS アプリケーションを用いた子供を見守るサービス) に使用される iOS アプリケーションへの新規機能追加。ウォーターフォール開発モデルで要件定義、基本設計、詳細設計、実装、単体テスト、結合テスト、保守の工程を担当。
    - アプリケーション コードのビルドを行い、顧客指定の形式に出力を行い、納品する。

- 業務スキル
    - 実装する機能を分担し、チーム内でどのように機能追加を行うかをディスカッション後、実装を行う。
    - 既存コード上に実装してあるアーキテクチャを理解し、既存関数の修正や新規関数の作成などを要件に合わせて実装。
    - 言語
        - 日本語: 日本語ネイティブである PM やチーム メンバーを中心に社内のメンバーとのコミュニケーションや技術的ディスカッション、協業を行えるレベル。

- 技術スキル
    - OS: iOS (12, 13)
    - API: REST API

- 開発スキル
    - 開発言語: Swift (5)
    - 開発モデル: ウォーターフォール

- プラットフォーム、ツール
    - API テスト ツール: Postman
    - プロジェクト管理: Redmine
    - エディタ/統合開発環境: Xcode, Vim

#### Android デベロッパー - Wi-Fi 認証テスト アプリ開発プロジェクト (2020 年 7 月 - 2020 年 11 月)
